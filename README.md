# Stock Analytica

A comprehensive stock analysis platform with AI-powered insights, interactive charts, and detailed valuation analysis.

## 🚀 New Features (Latest Update)

### 📈 Interactive Stock Price Charts
- **Google Finance-style charts** with time period selection (1D, 5D, 1M, 3M, 6M, 1Y, 3Y, 5Y, MAX)
- **Real-time price updates** with volume data
- **Interactive time window selection** buttons
- **Responsive chart design** with price and volume indicators
- **Chart highlighting capability** for future news correlation

### 💰 Enhanced Valuation Analysis
- **Relative Value Analysis**: Compare P/E, P/B, P/S ratios with peers
- **DCF Analysis**: Discounted Cash Flow calculations with sensitivity tables
- **Key Valuation Drivers**: Revenue growth, margin expansion, market share analysis
- **Peer comparison tables** with visual indicators
- **Comprehensive financial metrics** and ratios

### 📰 AI-Powered News Analysis
- **12-month news analysis** with emphasis on recent developments
- **Impact assessment** on stock price over next 12 months
- **News categorization** (earnings, regulatory, market, product, management, economic)
- **Confidence scoring** for news impact predictions
- **Filterable news timeline** by timeframe and impact type
- **AI summary analysis** of news impact on valuation

### 🎯 Chart Period Analysis (Future Feature)
- **Interactive chart highlighting** for specific time periods
- **News correlation** with price movements during highlighted periods
- **Event-driven analysis** explaining price volatility
- **Period-specific metrics** and performance indicators
- **Export capabilities** for highlighted periods

## 🛠️ Technology Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Charts**: Custom SVG-based interactive charts
- **State Management**: React Hooks
- **Routing**: React Router v6
- **Build Tool**: Vite

## 📦 Installation

```bash
# Clone the repository
git clone <repository-url>
cd stock-analytica

# Install dependencies
npm install

# Start development server
npm run dev
```
